About This Service

This is a demonstration service for testing mcp-vault features. It's a simple nginx container that requires secrets from HashiCorp Vault.

Configuration

‚ö†Ô∏è Secrets Required:
  • API_KEY - External service API key
  • DATABASE_URL - Database connection string
  • SESSION_SECRET - Session authentication secret

Testing Features

Use this service to test:

  • vault_scan_env - Scan the .env file for secrets
  • vault_scan_compose - Scan docker-compose.yml for secrets
  • vault_set - Store secrets to Vault with WebAuthn approval
  • vault_get - Retrieve secrets from Vault (tokenized)
  • vault_inject - Generate .env from Vault secrets

Quick Start

# 1. Scan for secrets
vault_scan_env(service="test-service")

# 2. Store to Vault (with WebAuthn approval)
vault_set(service="test-service", secrets={...}, approval_token="...")

# 3. Generate .env from Vault
vault_inject(service="test-service")

# 4. Start the service
docker-compose up -d
